Need help with the app, backups, purchases, catalog corrections, or missing images? Use the resources below.

## Helpful Details To Include

- Your device model and iOS version
- The app version you are using
- What you expected to happen
- What actually happened
- Screenshots if the issue is visual
- Whether the problem involves purchases, backups, photos, or catalog data

## Common Topics

### Catalog corrections

If an item is missing, mislabeled, or linked incorrectly, open an issue with the item name, type, and official source page if available.

### Backups and restores

Backups include collection status, wishlist, notes, and photos. If a restore seems incomplete, include the approximate date the backup was created and whether the file was exported from this app.

### Premium purchases

Monthly Premium and Lifetime Unlock are handled through Apple In-App Purchase. If a paid feature is still locked after purchase, use the in-app Restore Purchases option first, then report the issue if needed.

### Photos and notes

Photos and notes are stored locally on your device unless you choose to export them in a backup file.
